# Address Book 

I made this app in the subject "Programacion Web avanzada" as part of the bachelor degree "Ingenieria Informática" at [UOC](https://www.uoc.edu/portal/en/index.html)

### Description

The application is an address book where you can see, add and delete addresses. There are two different scenarios: when the user is logged in and when the user is anonymous.
- Anonymous user: they can see the list of public addresses
- Logged user: they can see all the addresses, add addresses and delete addresses.
